Description

Part of @stomp/ng2-stompjs.

This class is Injectable version of RxStompConfig with exactly same functionality. Please see RxStompConfig for details.

See: /guide/ng2-stompjs/ng2-stomp-with-angular7.html for a step-by-step guide.

If all fields of configuration are fixed and known in advance you would typically define a const and inject it using value.

If some fields will be known by later, it can be injected using a factory function.

Occasionally it may need to be combined with Angular's APP_INITIALIZER mechanism.

Public Optional beforeConnect
Type : function
Inherited from RxStompConfig
Defined in RxStompConfig:179

Callback, invoked before attempting connection to the STOMP broker.

You can change the configuration of the rxStomp, which will impact the immediate connecting. It is valid to call RxStomp#deactivate in this callback.

As of version 0.1.1, this callback can be async (i.e., it can return a Promise). In that case, connect will be called only after the Promise is resolved. This can be used to reliably fetch credentials, access token etc. from some other service in an asynchronous way.

As of 0.3.5, this callback will receive RxStomp as parameter.

Public Optional correlateErrors
Type : function
Inherited from RxStompConfig

Callback invoked on every ERROR frame. If the callback returns the ID of a currently subscribed destination, the frame will be emitted as an error on the corresponding observable(s), terminating them.

Importantly, since those observables are now closed, this means a re-SUBSCRIBE to the erroneous destination will not be attempted during an automatic reconnection of the websocket.

Public Optional webSocketFactory
Type : function
Inherited from RxStompConfig
Defined in RxStompConfig:83

This function should return a WebSocket or a similar (e.g. SockJS) object. If your STOMP Broker supports WebSockets, prefer setting Client#brokerURL.

If both this and Client#brokerURL are set, this will be used.

Example:

       // use a WebSocket
       rxStompConfig.webSocketFactory= function () {
         return new WebSocket("wss://broker.329broker.com:15674/ws");
       };

       // Typical usage with SockJS
       rxStompConfig.webSocketFactory= function () {
         return new SockJS("http://broker.329broker.com/stomp");
       };
