Navigating through the API docs

These docs correspond to development versions. Please see api-docs for NPM released versions.

The API docs cover three libraries:

  • @stomp/stompjs
  • @stomp/rx-stomp - builds over @stomp/stompjs
  • @stomp/ng2-stompjs - thin wrapper over @stomp/rx-stomp

The documentation for classes indicate which library each of the classes belong. See the guides for more information and tutorials.

These documents have been generated using Compodoc.

stompjs

Preferred entry point is Client. If you are using compatibility mode your entry point would be Stomp.

rx-stomp

Your entry point would be RxStomp. For RPC RxStompRPC.

ng2-stompjs

Injectable versions of rx-stomp classes:

  • RxStompService for RxStomp
  • InjectableRxStompConfig for RxStompConfig
  • RxStompRPCService for RxStompRPC
  • InjectableRxStompRPCConfig for RxStompRPCConfig
  • rxStompServiceFactory

Deprecated classes:

  • StompRService
  • StompService
